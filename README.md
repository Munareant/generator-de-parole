# Generator Simplu de Parole în Python

Acesta este un proiect simplu în Python care generează o parolă aleatorie pe baza lungimii specificate de utilizator.

## Scop

Proiectul a fost creat pentru a exersa conceptele de bază din Python, inclusiv:
- Funcții
- Module standard (`random`, `string`)
- Preluarea datelor de la utilizator (`input()`)
- Gestionarea erorilor simple (`try...except`)

## Cum se rulează

1. Asigură-te că ai Python instalat.
2. Clonează acest repository (sau descarcă fișierul `generator_parola.py`).
3. Rulează scriptul din terminal:

```bash
python generator_parola.py
